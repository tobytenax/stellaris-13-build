# Accessibility Statement

**Stellaris-13** is committed to ensuring digital accessibility for people with disabilities.

---

## Our Commitment

We are continually improving the user experience for everyone and applying relevant accessibility standards.

## Conformance Status

Stellaris-13 aims to conform to **WCAG 2.1 Level AA** standards.

### Measures Taken

- **Semantic HTML:** Proper heading hierarchy and landmark regions
- **Keyboard Navigation:** All interactive elements are keyboard-accessible
- **Color Contrast:** Text meets minimum contrast ratios (4.5:1 for normal text)
- **Focus Indicators:** Visible focus states for all interactive elements
- **Screen Reader Support:** ARIA labels where appropriate
- **Responsive Design:** Works across devices and screen sizes
- **Text Scaling:** Interface scales with browser text size settings

### Known Limitations

1. **Chart Wheel Visualization:** The SVG chart wheel may not be fully accessible to screen readers. We provide text-based placement tables as an alternative.

2. **AI-Generated Content:** AI interpretations may vary in structure and may not always follow optimal heading hierarchy.

3. **Third-Party Content:** External fonts and libraries may have their own accessibility considerations.

## Feedback

We welcome feedback on the accessibility of Stellaris-13. Please let us know if you encounter barriers:

- Report issues through the project repository
- Describe the problem and the assistive technology you're using
- We will respond within 5 business days

## Technical Specifications

Stellaris-13 relies on the following technologies:

- HTML5
- CSS3
- JavaScript (ES6+)
- SVG for chart visualizations

These technologies are relied upon for conformance with WCAG 2.1.

---

## Assessment Approach

Stellaris-13 self-assesses accessibility through:

- Manual testing with keyboard navigation
- Color contrast analysis tools
- Screen reader testing (VoiceOver, NVDA)
- Automated testing tools (axe, WAVE)

---

## Compatibility

Stellaris-13 is designed to be compatible with:

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Screen readers (VoiceOver, NVDA, JAWS)
- Browser zoom up to 200%
- High contrast modes
