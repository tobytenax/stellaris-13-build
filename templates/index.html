What This Tool Does

Stellaris-13 computes natal charts using two independent methods for mapping planets to the 13 astronomical constellations recognized by the IAU, including Ophiuchus.

Standard 13-Sign maps tropical longitude to unequal constellation spans derived from the IAU date ranges. This is the method used by most 13-sign astrologers.

IAU Astronomical uses J2000.0 constellation boundaries with precession correction, matching rigorous astronomical calculations. This precision implementation is unique to Stellaris-13.

Planetary positions are computed using the Swiss Ephemeris (pyswisseph), the same engine used by professional astronomy software worldwide.
